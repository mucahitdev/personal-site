# Cara Membuka Aplikasi Apa Pun dari Widget Layar Utama atau Layar Terkunci di iPhone

**Jawaban singkat:** iOS tidak mengizinkan Anda membuka aplikasi tertentu dari widget bawaan hanya dengan satu ketukan, tetapi widget pembuka aplikasi khusus bisa. Aplikasi seperti [FolderMini](/foldermini) mengubah widget menjadi kisi ikon aplikasi asli — ketuk ubin mana pun dan aplikasi itu langsung terbuka, langsung dari Layar Utama atau Layar Terkunci Anda.

Jika Anda pernah berharap bisa mengetuk sebuah widget dan langsung masuk ke aplikasi — alih-alih menggesek berhalaman-halaman ikon atau membuka Pustaka Aplikasi — panduan ini untuk Anda.

---

## Mengapa widget iOS bawaan tidak bisa membuka aplikasi tertentu

Widget bawaan Apple dirancang untuk *menampilkan informasi* (cuaca, kalender, baterai) dan, saat diketuk, membuka aplikasi pemilik widget tersebut. Widget itu memang tidak pernah dimaksudkan untuk berfungsi sebagai pembuka aplikasi lain Anda. Itulah sebabnya Anda tidak bisa, secara langsung, membuat widget yang setiap ubinnya membuka aplikasi berbeda.

Solusinya adalah **aplikasi widget folder** — sebuah utilitas kecil yang menampilkan kisi ubin aplikasi di dalam widget dan membuka aplikasi yang sesuai saat Anda mengetuk sebuah ubin.

## Langkah demi langkah: membuka aplikasi dari widget

1. **Pasang aplikasi widget folder.** Unduh widget pembuka aplikasi seperti [FolderMini](/foldermini) dari App Store. Gratis.
2. **Buat folder.** Buka aplikasinya dan tambahkan aplikasi yang ingin Anda akses dengan cepat — kelompokkan berdasarkan rutinitas, proyek, atau suasana hati.
3. **Tambahkan widget ke Layar Utama Anda.** Tekan lama area kosong pada Layar Utama, ketuk **+** di pojok atas, cari aplikasinya, lalu pilih ukuran widget (kecil, sedang, atau besar).
4. **Atau tambahkan ke Layar Terkunci Anda.** Tekan lama Layar Terkunci, ketuk **Sesuaikan**, lalu tambahkan widget aksesori untuk aplikasi yang paling sering Anda gunakan.
5. **Ketuk untuk membuka.** Setiap ubin menampilkan ikon aplikasi asli. Satu ketukan langsung membuka aplikasi tersebut.

## Apa yang perlu diperhatikan pada widget pembuka aplikasi

| Fitur | Mengapa penting |
| --- | --- |
| Ikon aplikasi asli | Ubin yang benar-benar Anda kenali, bukan placeholder generik |
| Pembukaan satu ketukan | Aplikasi langsung terbuka — tanpa menu perantara |
| Dukungan Layar Terkunci | Aplikasi utama Anda dapat diakses tanpa harus membuka kunci ke Layar Utama |
| Beragam ukuran | Muat 2 aplikasi atau 20, tergantung ukuran widget |
| Gratis dicoba | Tanpa komitmen untuk menguji apakah cocok dengan alur kerja Anda |

[FolderMini](/foldermini) mencakup semua ini: folder khusus warna-warni, ikon asli yang tepat pixel demi pixel, pembukaan satu ketukan, serta widget Layar Utama dan Layar Terkunci dalam ukuran kecil, sedang, dan besar.

## Pertanyaan umum

**Bisakah membuka aplikasi langsung dari widget di iPhone?**
Bisa — dengan widget pembuka aplikasi seperti FolderMini, mengetuk sebuah ubin langsung membuka aplikasi tersebut.

**Bisakah saya menaruh pembuka aplikasi di Layar Terkunci?**
Bisa. iOS mendukung widget aksesori Layar Terkunci, dan FolderMini memungkinkan Anda menempatkan aplikasi yang paling sering digunakan di sana.

**Apakah ini berfungsi tanpa pintasan atau otomatisasi?**
Ya. Aplikasi widget folder menangani pembukaan aplikasi secara native, jadi Anda tidak perlu membuat apa pun di aplikasi Pintasan.
