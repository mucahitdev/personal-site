# iPhone 홈 화면·잠금 화면 위젯에서 원하는 앱 바로 실행하는 방법

**간단한 답:** iOS는 기본 위젯에서 특정 앱을 한 번의 탭으로 열 수 없지만, 전용 앱 실행 위젯을 쓰면 가능합니다. [FolderMini](/foldermini) 같은 앱은 위젯을 실제 앱 아이콘 격자로 바꿔 줍니다 — 어떤 타일이든 탭하면 홈 화면이나 잠금 화면에서 곧바로 해당 앱이 열립니다.

아이콘 페이지를 넘기거나 앱 보관함을 열지 않고, 위젯만 탭하면 바로 앱으로 넘어갔으면 좋겠다고 생각한 적이 있다면 이 가이드가 도움이 될 것입니다.

---

## 기본 iOS 위젯이 특정 앱을 실행하지 못하는 이유

Apple의 기본 위젯은 *정보를 보여 주기* 위해 설계되었으며(날씨, 캘린더, 배터리), 탭하면 그 위젯을 소유한 앱이 열립니다. 다른 앱들을 여는 실행기로 쓰이도록 만들어진 것이 아닙니다. 그래서 별도 도구 없이는 각 타일이 서로 다른 앱을 여는 위젯을 만들 수 없습니다.

이를 해결하는 방법이 바로 **폴더 위젯 앱**입니다 — 위젯 안에 앱 타일 격자를 표시하고, 타일을 탭하면 해당 앱을 열어 주는 작은 유틸리티입니다.

## 단계별: 위젯에서 앱 실행하기

1. **폴더 위젯 앱을 설치하세요.** App Store에서 [FolderMini](/foldermini) 같은 앱 실행 위젯을 다운로드합니다. 무료입니다.
2. **폴더를 만드세요.** 앱을 열고 빠르게 접근하고 싶은 앱들을 추가합니다 — 루틴, 프로젝트, 기분에 따라 묶어 보세요.
3. **홈 화면에 위젯을 추가하세요.** 홈 화면의 빈 공간을 길게 누르고, 위쪽 모서리의 **+** 를 탭한 뒤 앱을 검색하고 위젯 크기(소형, 중형, 대형)를 선택합니다.
4. **또는 잠금 화면에 추가하세요.** 잠금 화면을 길게 누르고 **사용자화**를 탭한 뒤, 자주 쓰는 앱을 위한 보조 위젯을 추가합니다.
5. **탭해서 실행하세요.** 모든 타일에 실제 앱 아이콘이 표시됩니다. 한 번 탭하면 해당 앱이 바로 열립니다.

## 앱 실행 위젯에서 확인할 점

| 기능 | 왜 중요한가 |
| --- | --- |
| 실제 앱 아이콘 | 밋밋한 자리표시자가 아니라 실제로 알아볼 수 있는 타일 |
| 한 번의 탭으로 실행 | 중간 메뉴 없이 앱이 바로 열림 |
| 잠금 화면 지원 | 홈 화면까지 잠금을 풀지 않아도 주요 앱에 접근 가능 |
| 다양한 크기 | 위젯 크기에 따라 앱 2개부터 20개까지 배치 |
| 무료로 사용 | 내 워크플로에 맞는지 부담 없이 테스트 |

[FolderMini](/foldermini)는 이 모든 것을 지원합니다: 다채로운 맞춤 폴더, 픽셀 단위로 정확한 실제 아이콘, 한 번의 탭으로 실행, 그리고 소형·중형·대형 크기의 홈 화면 및 잠금 화면 위젯을 제공합니다.
